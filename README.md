# ⚡ Pokédex App

> A modern Pokédex built with React. Search Pokémon by name or ID, filter by type, and explore the full roster powered by the PokéAPI.

🌐 **[Live Demo →](https://pokedex-wilker.netlify.app)**

## ✨ Features

- 🔍 **Search** — Find any Pokémon instantly by name or ID
- 🏷️ **Filter by type** — Browse by Water, Fire, Grass, and more
- 📱 **Fully responsive** — Optimized for desktop and mobile
- ⚡ **Fast** — Powered by Vite and the PokéAPI

## ⚙️ Installation

### Prerequisites

- Node.js >= 18.0.0
- npm

### Steps

**1. Clone the repository**

```bash
git clone https://github.com/your-user/pokedex-app.git
cd pokedex-app
```

**2. Install dependencies**

```bash
npm install
```

**3. Start the development server**

```bash
npm run dev
```

**4. Open in your browser**

```
http://localhost:5173
```
